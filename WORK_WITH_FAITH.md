## Best-Fit Engagements

### Local AI Workflow Setup

For operators who want AI help without giving up local control, files, receipts, approvals, or production boundaries.

Outcome: a local workflow map, task packet structure, receipt folders, safety rules, and an operator walkthrough.

### FVE Website Visual Operations

For public sites with weak visuals, repeated assets, broken images, generic AI art, line-drawn placeholders, or inconsistent brand signals.

Outcome: review queue, keep/change recommendations, replacement plan, source receipts, and guarded publish-readiness handoff.

### GitHub / Public Surface Packaging

For private tools that need a credible public GitHub surface without publishing private source.

Outcome: public-safe README, docs, ownership posture, security language, launch receipt, and commercial CTA alignment.

### AI Brand System Audit

For brands whose public surfaces do not yet match the strength of the underlying work.

Outcome: brand-fit review, visual language notes, accessibility review, and next-batch creative direction.

### Protected File / Provenance Consulting

For creators and technologists preserving original files, proof records, authorship, and disclosure boundaries.

Outcome: provenance workflow, local receipt strategy, and public/private data boundary.
